# **App Name**: Avant Garde Boutique

## Core Features:

- Header with Unique Navigation: A header component featuring the brand name 'ZARA' and unique navigation links: Studio, Collections, Edit, and Sale.
- Editorial-Style Hero Section: A hero section with an original headline, a striking fashion banner, and a clear 'Explore Edit' call-to-action button.
- Signature Collection Display: Showcase three featured collection cards with a distinct layout, emphasizing the brand's unique style. The layout and content must feel original.
- Product Grid: Display a grid of eight static products with details (id, name, category, price, image) and 'Add to Cart' buttons.
- Add to Cart Functionality (Context API): Implement a cart context using React Context API to manage cart items: add, increase quantity, decrease quantity, and remove items.
- Cart Page/Drawer: Display cart items with quantity controls, remove options, and total price calculation.
- Brand Philosophy Section: A section containing a short paragraph detailing the brand's vision of modern fashion and sustainability.

## Style Guidelines:

- Primary color: Soft beige (#F5F5DC) for a calm, luxurious feel.
- Background color: Off-white (#FAFAFA), a slightly desaturated tint of the primary, will provide a subtle backdrop that allows products to stand out.
- Accent color: Light gray (#D3D3D3), a muted shade of gray that is close in hue to the primary but significantly less saturated, provides a neutral contrast that supports the overall minimalist aesthetic.
- Headline font: 'Playfair', a modern serif to capture a fashionable, high-end feel; Body font: 'PT Sans', a humanist sans-serif that is highly readable, suitable for longer passages
- Simple line icons for navigation and cart functions.
- Use large spacing to maintain a clean, open aesthetic.
- Subtle hover effects on buttons and product cards.
